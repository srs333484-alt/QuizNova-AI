SMART AI QUIZ GENERATOR



DESCRIPTION:

Smart AI Quiz Generator is a unique Python-based mini project that automatically asks quiz questions, checks answers, calculates scores, performs AI-style analysis, and displays results using an attractive terminal interface.



FEATURES:

1\. AI Performance Analysis

2\. Difficulty Levels

3\. Category Selection

4\. Random Questions

5\. Timer Tracking

6\. Negative Marking

7\. Grade System

8\. Motivational Quotes

9\. Colored Terminal UI

10\. Leaderboard System

11\. Replay Option



TECHNOLOGIES USED:

\- Python

\- Colorama Module



CONCEPTS USED:

\- Lists

\- Dictionaries

\- Loops

\- Conditions

\- Functions

\- File Handling

\- Random Module

\- Time Module



HOW TO RUN:

1\. Install Python

2\. Open terminal

3\. Run command:



python main.py
